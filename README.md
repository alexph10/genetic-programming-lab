# Genetic Algorithms Lab

A hands-on repository for studying and implementing core concepts from *Hands-On Genetic Algorithms with Python* by Eyal Wirsansky.

The goal of this repository is to build genetic algorithm concepts step by step through code, notes, and small experiments, with an emphasis on clarity and long-term reference.

## Current Focus

- Crossover methods
- Basic genetic algorithm operators
- Python implementations for study and experimentation
