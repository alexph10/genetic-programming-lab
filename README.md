# Genetic Algorithms Lab

A hands-on repository for studying and implementing core concepts from *Hands-On Genetic Algorithms with Python* by Eyal Wirsansky.

The goal of this repository is to build genetic algorithm concepts step by step through code, notes, and small experiments, with an emphasis on clarity and long-term reference.

## Current Focus

- Crossover methods
- Basic genetic algorithm operators
- Python implementations for study and experimentation

## Setup

```bash
git clone https://github.com/<your-username>/genetic-algorithms-lab.git
cd genetic-algorithms-lab
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

On Windows PowerShell:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Usage

Run a script directly with Python:

```bash
python ga_crossover_methods.py
```

## Notes

This repository is intended for learning, experimentation, and future reference. Some implementations may prioritize readability over optimization.
